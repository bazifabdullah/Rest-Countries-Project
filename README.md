## Overview

This a responsive web app that displays information about countries using <a href="https://restcountries.com/">REST Countries API</a>. Its a solution for a challenge by <a href="https://www.frontendmentor.io/home">Frontendmentor.io</a>. Built with React, it allows users to search for countries, filter by region, and view detailed information about each country. This project was built as part of my React learning journey to practice API fetching, Routing and State Management.

## Live Preview

<a href="https://react-rest-countries01.netlify.app/" target="_blank"> Click to view </a>

## Features

- Fetches and displays data from the REST Countries API.
- Search countries by nameFilter countries by region.
- View detailed country information (flag, population, subregion, capital, etc.).
- Navigate to bordering countries dynamically.
- Dark mode support.
- Responsive design.

## Built With

- React (Functional components + Hooks)
- Vite
- React Router v6
- Axios
- Tailwind CSS
- REST Countries API

## What I Learned

Working on this project helped me strengthen my understanding of React and its core concepts like:

- Implimenting functionality using React hooks (useState, useEffect, useParams).
- Fetching and managing data from API using React hooks.
- Using React Router for dynamic routing.
- Conditional Rendering (error/loading states).
- Deploying a React app on Netlify.

## Installation & Setup

- 1. Clone this repository
     git clone https://github.com/bazifabdullah/Rest-Countries-Project.git

- 2. Clone this repository
     cd Rest-Countries-Project

- 3. Install dependencies
     npm install

- 4. Start the development server
     npm run dev
